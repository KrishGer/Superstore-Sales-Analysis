 Superstore Sales Analysis (End-to-End Data Project)

🔥 Overview

This project performs an end-to-end analysis of a retail superstore dataset using SQL, Python, and Power BI.
The goal is to extract meaningful business insights and present them through an interactive dashboard.


🛠 Tech Stack
	•	SQL (SQLite) – Data querying & transformation
	•	Python (Pandas, Matplotlib) – Data cleaning & exploratory analysis
	•	Power BI – Interactive dashboard & visualization

 Superstore-Sales-Analysis/
│
├── data/        → Dataset used for analysis  
├── sql/         → SQL queries  
├── python/      → Jupyter notebook  
├── powerbi/     → Power BI dashboard  
├── images/      → Dashboard screenshot  
└── README.md

📊 Key Insights
	•	📈 Sales trends over time
	•	🌍 Regional performance comparison
	•	🛒 Top-performing products and categories
	•	👥 Customer behavior analysis
	•	💰 Average order value insights

 🚀 Key Features
